Things You Can Do Here

Say Hello

"Anything" sent to /hello/anything will say hello to "anything" so long as "anything" is a continuous word. For example,Say Hello to Elliot.

Upload a File...

Go to /upload_log to upload a file and have its stats detailed in the log. Alternatively, go to /upload_save to have the file written to the file system.
